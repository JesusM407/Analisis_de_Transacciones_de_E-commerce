# Analisis_de_Transacciones_de_E-commerce
este ejercicio, realicé un ciclo completo de análisis de datos (Pipeline ETL) sobre un dataset de más de 500,000 registros de ventas.

1) Limpieza y Normalización de Datos: Utilicé Python (Pandas) para transformar formatos de fecha complejos, gestionar valores nulos y normalizar nombres de columnas, asegurando la integridad de los resultados.

2) Gestión de Cancelaciones: Implementé lógica de filtrado para separar transacciones de ventas reales de devoluciones, permitiendo un cálculo preciso del Revenue (Ingresos).

3)  Análisis de Series Temporales: Creé visualizaciones con Matplotlib y Seaborn para identificar tendencias de ventas mensuales, detectando picos de demanda estacionales.

4)  Optimización de Inventario: Desarrollé un ranking de productos por rentabilidad, permitiendo identificar el "Top 10" de artículos que más impacto tienen en la facturación total.


Impacto: Este análisis permite a los niveles gerenciales entender no solo qué productos se venden más en volumen, sino cuáles impulsan realmente el crecimiento financiero y en qué periodos del año se debe reforzar el stock.
